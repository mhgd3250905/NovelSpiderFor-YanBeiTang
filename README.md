# NovelSpiderFor-YanBeiTang
这是一个专门用于爬取图书网站雁北堂（http://www.ebtang.com）的小爬虫，可以将网站上的书籍以txt格式保存到本地



> 作为一个推理迷，一直想看一下被称为国内新本格推理巅峰的作品《岛田流杀人事件》，苦于资源稀少，找了好久都找不到，最终找到雁北堂，不过必须在web或者app上看，所以动动手，写一个爬虫，爬下来自己转格式，想去哪儿看就去哪儿看。
